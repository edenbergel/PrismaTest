# PrismaTest

## Installation
```
npm install
npm run start:dev
```

### Mode production
```
npm run build
http-serve
```

### Choix techniques
* HTML5 / SCSS / JS
* webpack / Babel

### Fonctionnalités du projet
* Parallax sur plus d'une section
* Réutilisabilité du JS
* Menu fixe au scroll
* Responsive (Mobile First)
* Bloc Pub en sticky
